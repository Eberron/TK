🔑 DeepSeek API 密钥配置教程

💡 为什么需要配置API密钥？

TK智能总结插件使用DeepSeek AI来生成内容总结。为了保护用户隐私和控制成本，我们采用"用户自带API密钥"的方式：

  • ✅ 隐私保护：您的API密钥完全由自己控制
  • ✅ 成本透明：直接使用您的DeepSeek账户额度
  • ✅ 免费额度：DeepSeek提供充足的免费使用额度
  • ✅ 数据安全：密钥仅保存在您的本地浏览器中

📋 详细配置步骤

1注册DeepSeek账号

访问 https://platform.deepseek.com

点击右上角"Sign Up"按钮，使用邮箱注册账号。如果已有账号，直接登录即可。

2进入API管理页面

登录后，在左侧导航栏找到并点击 "API Keys" 菜单。

3创建新的API密钥

在API Keys页面，点击 "Create API Key" 按钮。

为您的密钥起一个容易识别的名称，比如"TK智能总结插件"。

4复制API密钥

创建成功后，系统会显示您的API密钥（以"sk-"开头的长字符串）。

⚠️ 重要提醒：

API密钥只会显示一次！请立即复制并妥善保存。如果忘记了密钥，需要重新创建。

5在插件中配置密钥

回到TK智能总结插件：

  1. 点击插件弹窗中的 "🔑 API设置" 按钮
  2. 将复制的API密钥粘贴到输入框中
  3. 点击 "💾 保存设置" 按钮
  4. 点击 "🧪 测试连接" 验证配置是否正确

🔧 常见问题解答

❓ DeepSeek免费额度够用吗？

DeepSeek为新用户提供充足的免费额度，通常包括：

  • 新用户注册即送免费tokens
  • 每月还有一定的免费额度
  • 对于日常的网页总结需求，免费额度通常足够使用

❓ API密钥安全吗？

是的，非常安全：

  • 密钥仅保存在您的本地浏览器中
  • 我们不会收集或存储您的API密钥
  • 所有API调用都是直接从您的浏览器发送到DeepSeek
  • 您可以随时在DeepSeek控制台中撤销或重新生成密钥

❓ 配置后还是不能使用怎么办？

请检查以下几点：

  1. 确认API密钥格式正确（以"sk-"开头）
  2. 确认密钥没有多余的空格
  3. 在设置页面点击"测试连接"查看具体错误信息
  4. 检查DeepSeek账户是否还有可用额度
  5. 确认网络连接正常

❓ 忘记了API密钥怎么办？

如果忘记了API密钥：

  1. 登录DeepSeek控制台
  2. 进入"API Keys"页面
  3. 删除旧的密钥（如果还记得名称）
  4. 创建新的API密钥
  5. 在插件中重新配置新密钥
